# Data README

This folder contains deterministic synthetic data for a public service operations analytics portfolio artifact.

The data is generated by `scripts/score_operating_data.py` with a fixed random seed. It models a health plan service operations environment with member services, claims operations, utilization management, provider services, billing and enrollment, pharmacy operations, care navigation, and risk adjustment support.

## Files

- `service_segments.csv`: segment-level metadata, business unit, stakeholder, baseline volume, financial sensitivity, utilization sensitivity, and avoidable-contact baseline.
- `weekly_operating_metrics.csv`: 20 weeks of service operations metrics for each segment.
- `dashboard_requirements.csv`: stakeholder requests translated into dashboard enhancement requirements.
- `data_quality_checks.csv`: source-system checks used to decide whether metrics are ready for recurring reporting.
- `initiative_monitor.csv`: action tracking for top service segments.

The values do not represent real company, member, provider, claim, utilization, or financial performance.

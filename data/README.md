# Data Sources

Synthetic source-style data for health plan service operations analytics.

- `entities.csv`: 36 service inefficiency segment records.
- `daily_metrics.csv`: 5,040 service inefficiency segment-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
